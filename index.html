ウェブサイト

自分のポートフォリオサイト

サイト説明

自分のポートフォリオサイトを作成しました。
私がどういう人間か知って欲しかったので、まず私自身の情報を載せました。
制作物においては、見やすいように１ページにまとめました。
１ページという縦に長いウェブサイトなので右下にボタンをつけていつでもトップに戻れるよう、固定ボタンにしました。
レスポンシブデザインモードでは、メニュー画面に場所を取られないようハンバーガーメニューボタンを作りました。
JavaScriptでの設定、メニューを開くと画像の最背面になってしまい、訂正するのが難しかったです。
色に関しては私自身明るい性格なのではつらつとしたイメージで黄色をメインカラーにしました。
また、少し優しい印象にしたかったのでCSSアニメーションでふんわり出てくるよう設定しました。

制作期間：3日間
使用言語：HTML　/　CSS　/　JavaScript　/　photoshop

医療法人こころ会　愛守病院

サイト説明

架空の病院ホームページを作成しました。
ターゲットは、30～60歳台の男女
ボタンや文字は見やすいように大きくしました。（老眼の方にとっても見やすいように） 情報量が多いと見にくいのでシンプルにまとめました。
シンプルにまとめるためのページ数や情報量の選定が特に難しかったです。
病院のwebサイトなので必要な情報は探さなくていいように載せる場所を考えました。
施設情報など表作成を沢山したのですが少し難しかったです。
色に関しては優しい雰囲気、信頼、清潔感のイメージで緑と青にしました。

デザイン：デザインカンプ

制作期間：５日間
使用言語：HTML　/　CSS　/　JavaScript　/　jQuery 　/　photoshop

アロマトリートメントサロン

サイト説明

友人のアロマトリートメント店のWEBサイトを作成しました。
ターゲット層は20～40歳台までの女性。
こだわった点は、現在営業中か閉店中かを分かりやすい部分に表示しお客様に利用しやすいよう工夫しました。
また、キャッチフレーズなど特に希望はないとのことでしたので、キーワードを洗い出し、作成したことが特に大変でした。
料金に関して妥協はしたくないとのことだったので、最終的に色も少し落ち着いた、リッチ感を出せるような色を選びました。また、ターゲットは女性なので優しい 今後打ち合わせをし、友人とデザインのすり合わせをしていく予定です。

デザイン：デザインカンプ

制作期間：４日間
使用言語：HTML　/　CSS　/　JavaScript　/　photoshop

ロゴ・ポスターデザイン

猫のロゴ　作成意図

友人が飼っている猫のロゴを作成しプレゼントしました。
猫はすごくわんぱくで元気な雄の子猫なので、Rから顔を覗かせお茶目感を演出してみました。 カラーは顔が可愛いのでピンクで、名前のロゴは男の子っぽく緑にしてみました。
猫を切り抜き、Rの間から出てくる感じを作るのが難しかったです。

使用ソフト：photoshop

旅行会社のポスター　作成意図

学校のデザインの課題授業で制作しました。
あらかじめ写真が何枚か、ロゴ画像、キャッチフレーズや文章などは準備されており、それを使用しそれぞれデザインしポスターを制作するというものでした。
コンセプトは、幅広い年齢の男女で鹿児島中央駅に飾られているポスターという設定にしました。
鹿児島県は雪が珍しく銀世界をみてみたいという思いがあるのではと考え、人際目を引く銀世界の写真を選びました。
キャッチフレーズの一つに『まだみたこのない景色』などの言葉が含まれているものを選びました。（銀世界はまだみた事のない人たちが多いと思うので）
しかし写真のインパクトを出すために字は小さく真ん中に載せました。
『日本』という文字は日本の国旗をイメージし赤に、また国内旅行を強調したかったのでこのカラーを選びました。

使用ソフト：PowerPoint
